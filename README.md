A sleek, neon-themed task management application designed with a futuristic UI and advanced productivity features. This app goes beyond basic list-making by offering dynamic sorting, progress tracking, and file portability.

# ‚ú® Key Features
Advanced Task Management: Create, edit, and delete tasks with ease. The edit mode intelligently loads existing data for quick modifications.
Dynamic Progress Tracking: A real-time glowing progress bar visualizes your completion rate.
Intelligent Reminders: * Visual Cues: Overdue tasks are highlighted with a distinctive red glow.
Alerts: Native notifications for overdue tasks and items due within the hour.
Smart Sorting & Filtering: * Multi-Criteria Sorting: Sort by Due Date, Category, or Priority (mapped numerically: High=3, Medium=2, Low=1).
Real-time Search: Filter tasks by text, category, or priority level as you type.
Data Portability: * Export: Save your tasks to a JSON file.
Import: Restore tasks from a JSON file with built-in format validation.

# Immersive UI/UX:
Dark Mode: A persistent dark/light theme toggle stored in localStorage.
Smooth Animations: Tasks slide and fade during actions; buttons feature ripple effects and hover glows.
Keyboard Navigation: Fully accessible via Tab, Arrow keys, and Enter.
